# slack-better-bitbucket

A better Bitbucket incoming webhook for Slack.


## Options

    Usage: slack-better-bitbucket [options]

    Options:
        --slack-incoming-uri URI                    URI for Slack incoming webhook
    -p, --port PORT                   8880          Port number
        --dev                                       Development mode
        --keystore PATH               keystore.jks  Keystore to use for SSL certificates
        --keystore-password PASSWORD                Keystore password
    -h, --help
